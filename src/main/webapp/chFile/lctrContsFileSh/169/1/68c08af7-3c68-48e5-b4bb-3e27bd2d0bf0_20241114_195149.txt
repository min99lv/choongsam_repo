[ 기능 ]

- 수강생 기준 출결현황 조회 ( 출석, 지각, 결석, 출석율 )
- 강사 기준 출결현황조회 ( 한 강의의 총 수강생의 가장 최근차시의 (출석수, 지각수, 결석수, 출석율))
- 강사 기준 차시별 출석 조회(이름, 출석, 지각, 결석)
- 수강생 기준 과제 조회 (강의명(수강중인 강의리스트), 진행여부), 첨부파일 다운로드
- 수강생 기준 과제 제출 첨부파일 업로드
- 강사 기준 과제 제출 조회 ((과제시작일과 마감일에 따라 과제 진행상태 - 미등록/진행중/종료), 한 강의의 총 수강생의 제출율)
- 강사 기준 과제 등록 첨부파일 업로드
- 수강생 기준 성적 조회 (출석점수와 과제점수로 학점 계산)
- 강사 기준 성적 등록 및 수정 (수정버튼 클릭시 ajax로 총출석점수 혹은 총과제점수 혹은 둘다 수정, 수정된 총 출석점수와 총 과제점수로 학점 업데이트)

수강생  기능 목록
- 수강생 기준 출결현황 조회
  출석, 지각, 결석, 출석률 정보 제공

- 수강생 기준 과제 조회
  수강 중인 강의 리스트와 과제 진행 여부 제공
  첨부파일 다운로드 기능 포함

- 수강생 기준 과제 제출
  첨부파일 업로드 기능 제공

- 수강생 기준 성적 조회
  출석 점수와 과제 점수로 학점 계산 기능 제공


강사 기능 목록
- 강사 기준 출결현황 조회
  한 강의의 총 수강생의 가장 최근 차시 기준
  출석수, 지각수, 결석수, 출석률 제공

- 강사 기준 차시별 출석 조회
  학생 이름, 출석 여부, 지각 여부, 결석 여부 제공

- 강사 기준 과제 제출 조회
  과제 시작일과 마감일에 따른 진행 상태 (미등록/진행중/종료) 제공
  한 강의의 총 수강생의 제출율 제공

- 강사 기준 과제 등록
  첨부파일 업로드 기능 제공

- 강사 기준 성적 등록 및 수정
  수정 버튼 클릭 시 AJAX로 출석 점수 및 과제 점수 수정
  수정된 점수로 최종 학점 업데이트 기능 제공
